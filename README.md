# Prison Management System
A Java OOP project developed by a Software Engineering student team.

## Project Idea
The system manages:
- Prisoners
- Guards
- Lawyers
- Crimes
- Prison Cells

## OOP Concepts Used
- Inheritance
- Polymorphism
- Method Overriding
- Composition
- Association
- Encapsulation

## Class Structure
- Person (Base Class)
- Prisoner
- Guard
- Lawyer
- Crime
- Cell
